# Needs/Features List

<br/>


### Needs

```
1. Mobile friendly 
2. Keeping personal/voting data secure/private 
3. Database to store voting information with network access
4. Accessible through web
```


### Features
```
1. Create a vote 
2. View results of votes
3. User accounts / authentication
4. View and then cast a vote on a vote that has been created
5. View analytics of a vote 
6. Google cloud ai TTS and STT
7. Cast an anonymous vote
8. Make sure votes are unique
9. Share vote with others link or code
10. Different kinds of votes (ranked voting as an option)
```
